# Year 2 - Advanced Foundations

Weekly Commitment: 30-35 hours

## Phases

### Phase 1: Advanced Linear Algebra & Numerical Methods (Weeks 1-12)
- Matrix Decompositions (LU, QR, SVD)
- Vector Spaces & Inner Product Spaces
- PCA Applications
- Numerical Linear Algebra
- Optimization I & II

### Phase 2: Probability, Statistics & Stochastic Processes (Weeks 15-25)
- Multivariate Probability
- Random Variables & Transformations
- Markov Chains
- Regression & Estimation
- Statistical Simulation

### Phase 3: Advanced CS Fundamentals & Algorithms (Weeks 28-38)
- Advanced Data Structures
- Algorithms II (DP, Greedy)
- Graph Algorithms
- Complexity & Algorithm Analysis

### Phase 4: Applied Mathematics + Intro Distributed Systems (Weeks 41-52)
- Linear Algebra + Probability Applied
- Algorithmic Problem Solving
- Intro to Distributed Systems
