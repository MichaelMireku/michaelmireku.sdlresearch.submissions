# Year 5 - Advanced Distributed Systems & Performance Engineering

Goal: Optimize distributed systems and ML pipelines; work on performance, security, and scalability

## Phases

### Phase 1: System Optimization (Weeks 1-12)
- Profiling, bottleneck analysis
- Concurrency control, load balancing

### Phase 2: Security & Reliability (Weeks 15-26)
- Verifiable computation
- Secure updates, fault injection
- Consistency models

### Phase 3: Large-Scale ML Integration (Weeks 29-40)
- Distributed training
- Federated learning
- Incremental weight updates

### Phase 4: Capstone Project (Weeks 41-52)
- End-to-end distributed ML pipeline
- Edge-ready with performance and security benchmarks
